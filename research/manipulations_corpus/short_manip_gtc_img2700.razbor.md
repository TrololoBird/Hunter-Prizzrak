## Индикаторы — НЕТ (чистый price action)
На всех ТФ график **без единого индикатора** — нет объёмного пейна, RSI/MACD/MA, хитмапа.
Разметка только ручная: горизонтальные уровни + восходящая трендовая/канал. **Edge
структурный** (импульс + снятие ликвидности + поглощение), не индикаторный → сканеру не нужен
индикаторный гейт для этого класса.

## Таймфрейм-иерархия и разметка
**1Д** (структура, уровни, восходящая трендовая от мартовского лоя) → **15м** (точка входа:
пик-свеча 01 июн 13:45, High 0.005418) → **30м** (нарисован **восходящий канал** — две

## Логика сетапа (аудио ⊕ видео)
1. **1Д:** восходящий канал с февраля, серия импульсов — каждый поглощался.
2. **Триггер:** ждать **финальный импульс = обновление максимумов** (снятие ликвидности
   сверху к 0.00507–0.00550). Когда максимум обновлён И ликвидности выше нет → набирать шорт.
3. **15м вход:** памп снял ликвидность (пик 0.005418) → свечи затухают → **красная импульсная
   свеча закрылась = хай сформирован** → шорт **от пика**.
4. **Управление:** ~30% чистого на шорте пампа; первая цель **0.00324**, далее возможный
   пробой к **0.0015**; раннер держат; стоп — за импульсный максимум (~0.0055).
5. **Топливо:** пул ликвидности под минимумами (стопы лонгистов с пампа).

## Причина → следствие (механистично + количественно)
- **Что реально произошло:** шорт от 0.005418 (01 июн) краткосрочно пошёл в плюс (→~0.0040,
  ~−26%, тут «30% чистого» реалистично). НО **12 днями позже, 13 июня, JCT перепампили до
  0.008272** — это **+52.7%** ПРОТИВ шортовой позиции. Только после этого — пролив к цели
  0.00324 (лоу 0.0031).
- **Следствие для риска:** стоп «за импульсный максимум» (~0.0055) был бы **снесён** пампом
  13 июня. Т.е. «раннер до 0.0034» без выхода на первой цели → стоп-аут, а не win. Чистый
  нарратив «зашортил памп — он слился» **скрывает +52.7% adverse excursion**.
- **Вывод:** прибыльна только **быстрая фиксация на первой ноге** (0.0055→0.0040) или
  ре-энтри; «пересиживание» первого импульса на манипуляции = смертельно (второй импульс
  манипуляции может быть больше первого — см. long-разбор «пропусти первый памп»).

## Данные для сканера (заземлённые 33 кадрами + OHLCV)
1. **Паттерн SHORT-манипуляции** детектируем: range-expansion бар берёт прежний максимум +
   нет ликвидности выше + разворотная импульсная свеча. Без индикаторов.
2. **⚠️ Риск-правило из данных:** первый «финальный импульс» **часто НЕ финальный** — 13 июня
   был второй, больше (0.0054→0.0082). → сканер должен либо (а) ставить тейк на первой
   структурной цели (0.00324) и НЕ держать раннер против возможного второго импульса, либо
   (б) требовать подтверждения слома структуры на младшем ТФ до шорта. Стоп «за максимум»
   недостаточен на памп-дамп инструментах.
3. **Метки исхода для бэктеста:** `target1_hit` (0.00324 — да), `target2_hit` (0.0015 — нет),
   **и `max_adverse_excursion`** (+52.7%) — MAE обязателен, иначе бэктест переоценит сетап.
   Ср. [[scanner-negative-expectancy-backtest]].
4. **MTF-гейт:** контекст 1Д, вход 15м; на 30м — восходящий канал для управления.
5. **Юниверс:** JCT есть на Binance USDⓈ-M — сетап виден штатным фидом.
