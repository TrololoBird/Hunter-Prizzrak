## Тезис и скелет
«Как читать LONG-пампы на сотни %. 3 самых надёжных вида, 5–6/мес, каждая >50% чистого.» Общий
скелет: агрессивный памп → **поглощение одной свечой** → боковик с накоплением ликвидности → крупный
игрок добирает/пампит для выноса шортистов → **вход в нижней части боковика только после слома
нисходящей структуры / закрепа выше макс + бычий объём**.

## Три вида (точные уровни/даты с высокоres-кадров)
- **Вид №1 «памп → поглощение одной свечой → редистрибуция» — ESPORTSUSDT (4ч).** База ~0.04 →
  вертикальный памп. **Первый памп (+934%, к ~0.80 в мае) НЕ торговал** (нет закрепа, ликвидность
  снизу). Торговал **следующую ногу** (13 июн, замер +359%): вход ~0.06 после поглощения/выноса вниз →
  памп к **0.3371**; заявлено +160% чистого.
- **Вид №2 «канал → манипуляция вниз → закреп над макс» — ZEREBROUSDT (4ч).** Восходящий канал → макс
  → манипуляция вниз → нисходящий канал → боковик. Триггер = **закреп выше пред. максимума на бычьих
  объёмах**. Вход **0.032**, замер **+45.98%** к **0.048767** (~22 июн); взял «всего 20%».
- **Вид №3 «нисходящий клин + накопление» — BSBUSDT (4ч).** Долгий даунтренд (памп в истории
  необязателен) → накопление у лоя ~**0.20** → лонг. «100% за вечер, среднесрок 250%», памп к **0.6884**.
- **Пробой уровня (доп) — HEIUSDT (1Д).** Лонг на пробой круглого **0.1** → лоу 0.05488 → пик **0.20274**.
- **Спот-пример — PLAYSOUTUSDT (15м).** Замер **+62.34%** 12 июн (0.029→0.049), пост академии 22:13.

## Причина → следствие (механистично + количественно)
- **Механика:** MM гонит цену вверх (сбор ликвидности сверху + FOMO), распределяется в импульс; после
  распределения бид иссякает. Первый импульс часто **не финальный** — крупняк добирает и пампит ещё раз,
  чтобы вынести шортистов (поэтому автор «пропускает первый памп»). Лонг берётся на **втором, надёжном**
  импульсе после слома нисходящей структуры.
- **Количество/выживаемость:** каждый фьючерс после пампа **полностью откатился** (ESPORTS −96%, BSB
  −81%, ZEREBRO −65% от пика). LONG-edge существует **только на памп-ноге с жёстким выходом на цели**;
  удержание = −65…−96%. Это НЕ «пересиживание».

## Данные для сканера
1. **Три детектируемых паттерна** (см. `scanner` A/B/A3/C): (№1) range-expansion памп → bearish-engulf
   базы → повторный памп; (№2) канал→манипуляция-вниз→**закреп выше макс + бычий объём** (триггер входа);
   (№3) нисходящий клин + накопление у лоя. Доп: чистый пробой круглого уровня (HEI).
2. **Триггер = слом нисходящей структуры / закреп над макс + объёмное подтверждение** — конкретное BOS+
   volume условие; проверить в state-машине.
3. **Метки исхода для бэктеста:** `pump_hit` (все ✅) + **`survivability`/round-trip** (все умерли) +
   **MAE**. Без метки «после сигнала цена вернулась ниже входа» бэктест переоценит LONG-манипуляции.
   Ср. [[scanner-negative-expectancy-backtest]].
4. **«Пропусти первый памп»** — эвристика: первый импульс с невыкупленной нижней ликвидностью ненадёжен;
   ждать второй. Кодируемо: не входить, пока нижняя ликвидность не снята + нет закрепа.
5. **Юниверс:** 4/5 на **Binance USDⓈ-M** — сканер видит; PLAYSOUT — Bybit-only. Все — свежие листинги
   (контракты с апреля'26): фильтр возраста не должен их выбрасывать.
