# HUNTER — два промта для Claude Code

**Промт 1 (Fable 5)** — глубокий аудит + проектирование конечной версии (только анализ/документы, код не менять).
**Промт 2 (Opus 4.8)** — исполнение одобренного, схождение кода к целевой версии.
Между ними — **твой гейт** (оцениваешь вывод Fable на уровне сигнала, одобряешь, потом запускаешь Opus).

> Калибровка живыми обзорами (видео/текст от «Призрак» и др.) — **следующая фаза, вне текущей задачи.** Fable проектирует версию, *готовую* к такой калибровке (калибровочная поверхность + точки подключения), но сам цикл с присланными обзорами сейчас не строится.

**Контекст для обоих:** `HUNTER` — крипто-фьючерс signal-analytics (Python 3.14 / Polars / CCXT public-only), **полностью сгенерирован ИИ** по методикам Telegram-каналов **«Призрак»**, **«Owner of Shorts»**, **«Спекулянтка»**. Владелец **не разработчик** — смотрит сигналы. Источник истины — **метод каналов** (`docs/`, `research/prizrak_corpus/`), НЕ код и НЕ статьи.
