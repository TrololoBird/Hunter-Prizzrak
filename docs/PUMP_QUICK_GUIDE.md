# Pamp hunter — экспресс-методика

## 3 типа манипуляций в лонг

### Тип 1: Агрессивный памп → поглощение → боковик → свип → репамп
```
1. Агрессивный памп вверх        — 4h, тело ≥ 1.5×ATR
2. Поглощение одной свечой       — 4h, цена возвращается на уровень ДО пампа (≥80% retrace)
3. Боковик-1                     — 4h/1h, ≥3 касания, ATR сжат ≤70%
4. Свип вниз (сбор стопов)      — 4h/1h, хвост ≥30%, закрытие внутри боковика
5. Боковик-2 (накопление)        — 4h/1h (более узкий, объём растёт)
6. BOS/CHoCH вверх               — 15m/5m, объём ≥ 1.5× среднего ← **точка входа**
7. TP1: хай боковика → TP2: хай пампа → TP3: следующий хай
```
**Ключевое правило**: «следующий памп надёжнее». Если нет чистой структуры — пропускай.

### Тип 2: Восходящий канал → свип вниз → нисходящий канал → боковик → пробой
```
1. Восходящий канал              — 4h/1d
2. Манипуляция вниз              — свип ниже канала
3. Нисходящий канал              — 4h/1h
4. Боковик (накопление)          — 4h/1h, ликвидность сверху и снизу
5. Закреп ВЫШЕ хая канала        — 4h/1h, обязательно с объёмом ≥ 1.5×
6. BOS/CHoCH                     — 15m → **вход**
```
**Ключевое**: без закрепа выше хая + объёма — цена уходит вниз.

### Тип 3: Нисходящий канал → накопление → пробой (без начального пампа)
```
1. Нисходящий канал (долгий)     — 1d/4h, постепенные HH/LL
2. Боковик (накопление)          — 4h/1h, ≥5 касаний (дольше — нет импульса)
3. Объём растёт к нижней границе
4. BOS/CHoCH                     — 15m → **вход**
5. TP: больше, чем Тип 1 (100-250%) — нет хая пампа сверху
```

## Контрольные точки
- Score ≥ 70% — полный вход, WS 1m подписка
- Score 50-69% — половинный лот, WS подписка
- Score 30-49% — наблюдение
- < 30% — нет паттерна

## Когда НЕ входить
- Нет закрепа выше хая (Тип 2)
- Свип без возврата в боковик
- Слом структуры без объёма (< 1.5× среднего)
- Аномальный объём в противоположную сторону (≥ 2×)
- Score < 50% без структуры
